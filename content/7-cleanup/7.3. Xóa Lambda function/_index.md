### 7.3. Xóa Lambda function {#xoa-lambda}

1. Truy cập **AWS Lambda**.  
2. Chọn function đã tạo (ví dụ: `masking-function`).  
3. Nhấn **Delete**.  
4. Nhập tên function để xác nhận.
